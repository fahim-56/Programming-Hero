Anchor

The anchor tag < a> is used to create hyperlinks in HTML. It allows you to link to other web pages, files, locations within the same page, email addresses, and more.

Attributes of the Anchor Tag

  • href: URL of the page the link goes to. It can be an absolute URL (full web address) or a relative URL (path relative to the current page).
  • target: This attribute specifies where to open the linked document. Common values include:
    • _self: Opens the link in the same frame (default).
    • _blank: Opens the link in a new tab or window.
    • _parent: Opens the link in the parent frame.
    • _top: Opens the link in the full body of the window.
